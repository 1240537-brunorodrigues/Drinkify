Como Utilizar o Menu

  1. Lançar o Dado: Clica no ícone do dado no canto superior esquerdo para lançares um dado digital (1–6). Usa este valor para avançares a tua peça no tabuleiro físico.
  2. Carta: Seleciona “Tirar Uma Carta” para revelar uma carta aleatória do baralho. Segue as instruções indicadas na carta.
  3. Pergunta: Seleciona “Pergunta” para veres uma pergunta de escolha múltipla. Se responderes bem, ganhas um benefício; se estiver errado, sofres uma penalização.

Regras Gerais

1. Os jogadores jogam por turnos, lançando o dado físico no tabuleiro ou utilizando o dado digital.
2. Ao cair numa casa colorida:
  • Branco = Nada acontece.
  • Azul = “Bebe um gole”.
  • Rosa = Tirar uma carta.
  • Roxo = Responder a uma pergunta.

Por favor, bebam com responsabilidade e divirtam-se!
